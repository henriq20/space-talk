# SpaceTalk

SpaceTalk is a forum where people can talk about our vast universe.

It contains all functionalities that a major forum does.

## Functionalities

On this website you can:

- <b>CRUD</b> Posts;
- Create/Read Comments;
- Upvote/Downvote Posts.
